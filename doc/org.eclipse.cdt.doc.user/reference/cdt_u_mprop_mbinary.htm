C/C++ Project Properties, Managed, C/C++ Build, Binary Parser

You can select the Binary Parsers you require for the project.

To ensure the accuracy of the C/C++ Projects view and the ability to successfully run and debug your programs. After you select the correct parser for your development environment and build your project, you can view the symbols of the object file in the C/C++ Projects view.

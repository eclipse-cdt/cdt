New Project Wizard - Standard Make, Error Parsers

You can select which error parsers to use and in which order they are used for your project.

Name Function
Error Parsers You can select which Error Parsers to enable from this window.
Up Moves the currently selected Error Parser higher in the ordered list.
Down Moves the currently selected Error Parser lower in the ordered list.
Select All Selects all Error Parsers.
Unselect All Unselects all Error Parsers.

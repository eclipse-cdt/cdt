Generating a Config File for the Formatter Application

Generating a config file for the formatter application involves modifying the code formatter settings for a C/C++ project and copying org.eclipse.cdt.core.prefs out of the .settings directory for that project.

  1. Select a C/C++ project, open the pop-up menu and choose Properties.
  2. Select the C/C++ General > Formatter page and check Enable project specific settings.
  3. Select or edit a profile as explained above.
  4. Click OK when you are done.
  5. Use either a file manager or the command line to copy workspace/YourCProject/.settings/org.eclipse.cdt.core.prefs to a new location.

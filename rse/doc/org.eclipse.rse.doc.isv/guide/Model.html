Remote System Explorer Model

The underlying containment model for the RSE static artifacts is shown here.

At runtime, the model takes a slightly different form:

  • Profiles are managed by a profile manager, and contain:
    • Host Pools, one per profile, which contain
      • Hosts
    • Filter pools, keyed by Subsystem Configuration, and managed by a filter pool manager which contain
      • Filters, which contain
        • Filter strings
  • Subsystem Configurations contain:
    • Subsystems, keyed by host, which contain
      • References to filter pools, which are managed by filter pool managers
